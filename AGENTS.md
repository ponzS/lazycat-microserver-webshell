# AGENTS.md

## 项目级约束

- 始终使用简体中文回应。
- 本项目禁止引入 `tmux`。
  - 不得把 `tmux` 添加为运行依赖、构建依赖、镜像/打包依赖或脚本依赖。
  - 不得在 Go、Shell、配置文件、构建文件或运行时逻辑中调用 `tmux`。
  - 不得基于 `tmux` 实现会话、pane、tab、复用、保活或终端管理能力。
- 本项目禁止引入 `xterm.js`。
  - 不得添加 `xterm`、`@xterm/*`、`xterm-addon-*` 等 npm 依赖。
  - 不得复制、内嵌、改名或 vendoring `xterm.js` 及其插件源码。
  - 不得基于 `xterm.js` API 设计或实现前端终端功能。
- 如果未来新增 WebShell、终端渲染、会话管理、tab/pane、保活或复用能力，必须遵守以上限制；如果某个方案依赖 `tmux` 或 `xterm.js`，需要先改用不依赖它们的方案，而不是绕过本规则。
- 参考外部示例代码时，如果示例包含 `tmux` 或 `xterm.js`，只能作为概念参考，不得把相关代码、依赖或调用路径带入本项目。
